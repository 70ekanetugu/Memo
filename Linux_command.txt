◆Linux　コマンド・ショートカット集
～基本コマンド～
	man "コマンド"　マニュアル表示
	pwd  print working directoryの略。環境変数?シェル変数?にある
	ls [オプション]　カレントディレクトリ内のファイル一覧表示
	file [オプション] "ファイル名" 　ファイル形式を調べるコマンド
	cd 　　　　　　カレントディレクトリの変更。./../~
	env [オプション]　　オプション無ければ環境変数の表示
	less/more 　ページごとに表示。パイプで使われる事が多い。
	grep "検索正規表現"　["ファイル名"]　　文字列検索コマンド	
		-i 大文字、小文字区別しないで検索
		-n 検索結果に行番号表示
	ファイル(コマンド)A [> / >> / 2> / 2>> / 2>&1]　ファイル(コマンド)B　リダイレクト
		>：結果をリダイレクト(上書き)
		>>：追記
		2>：エラー表示を保存
		2>>：エラー表示を追記
		>..2>&1：結果とエラーを保存
		>>..2>&1：結果とエラーを追記 	
[ファイル操作]
	mv [オプション] 元ファイル　新ファイル(又は変更後のファイル名orディレクトリ名)
	cp [オプション]　元ファイル　コピー先　 ディレクトリの場合は-rオプション
	dd ディレクトリのコピー
	mkdir [オプション]　ディレクトリ名 
		-p:階層付けして作成できる。(p無ければ1個ずつ作る必要がある)	
	rm [オプション] ファイル..     ディレクトリの場合は-rオプション

[ACL/ファイルパーミッション/ユーザー管理]
	getfacl "ファイル名"　ACL(Access Control List)エントリの表示
	setfacl [オプション]　"ACLエントリ" "ファイル/ディレクトリ名"　　ACLの追加、変更
	　・-m：ACLエントリの追加、変更の際、使う。
	　・-x：ACLエントリ削除
	　・--remove-all：すべてのエントリ削除
	　・ACLエントリ：getfaclで表示される権限付与者　	
	chown [オプション]　"user" "ファイル/ディレクトリ"　　所有者の変更
	chown "user":"グループ名" "ファイル/ディレクトリ"　　所有者とグループの変更	
	chgrp "グループ名"　"ファイル/ディレクトリ"	グループの変更
	chmod [オプション] "パーミッション" "ファイル/ディレクトリ"　ファイルのパーミッション変更(r w x)
	　<パーミッション指定方法>:①～③をつなげる
		①"u"/"g"/"o"/"a"：所有者/所有G/その他ノユーザ/全てのユーザ
		②"+"/"-"/"="　　：権限の追加/削除/指定
		③"r"/"w"/"x"　　：読込権限/書込み権限/実行権限
	id -a　ユーザー情報の表示
	useradd [オプション]　[ユーザー名]　ユーザーの追加
	　・-a:指定した補助グループにユーザーをちゅかする
	　・
	userdel　　ユーザーの削除		
	usermod　　ユーザーの修正
　　※GUIの場合、上記3つは"system-config-users"で行える。　
[圧縮・アーカイブ]
	tar [オプション]　[ファイル名...]
		f　ファイル名を指定
		c　新しいアーカイブを作成する
		x　アーカイブからファイルを抽出(展開)
		r　末尾に追加
		v　詳細を表示する
		j　bzip2ファイルを扱うことを指定
		z　gzip ファイルを扱うことを指定
	CPIO [モード指定]　3つのモードがある。

[プロセス管理]
	ps [オプション]　
		a:端末操作のプロセスを表示
		x:端末操作以外のプロセスを表示
		u:CPUやメモリの使用率も表示
		-A:全プロセスの表示
		-u:プロセスを実行しているユーザーを指定する
		-p:PIDを指定する
		--sort:プロセスの表示順を指定する。
		その他にもたくさんある。
	kill [-シグナル] "プロセスID"　　プロセスの終了(PID指定)
	killall [-シグナル] "プロセス名"　killと同様。プロセス名称で指定する場合
	service "サービス名"　start/stop/restart/status 　サービスの開始等　
	systemctl　[オプション] [COMMAND]　"サービス名"
		オプションはめったに使わない？
		コマンドは"start/stop/restart/status/enable/disable"等をよく使う。

Alt+b　　　　単語谷で左へ
Ctrl+Home　ページの先頭へ
Ctrl+End　　ページの最後へ
Tab　　　　コマンド、ファイル名の補完
Ctrl+l　　　画面のクリア
Ctrl+c　　　実行中プログラムの強制終了
Ctrl+s　　　画面更新の一時停止
Ctrl+q　　　画面更新の再開
